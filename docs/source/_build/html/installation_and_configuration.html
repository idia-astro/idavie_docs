Installation and configuration¶

Executable¶

Once the requirements described in System requirements, are installed and working correctly, the user can unzip the provided iDAVIE-v.zip, which contains the executables .exe (and other reference files).

To start the software, double-click on the executable iDaVIE-v file.

Troubleshooting¶

If errors are encountered that you can’t solve, please send us the log files along with your bug reports. The log files can be found in the directory %appdata%/../LocalLow/DefaultCompany/iDaVIE-v

(Press WIN + R to open the run dialog, and paste the above address in to open the folder).
